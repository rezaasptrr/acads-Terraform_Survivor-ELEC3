## Quick Start

```bash
# Play immediately (easiest)
python ../../run-game.py

# Or open directly
open ../../src/frontend/standalone.html

# With Terraform
cd ../../src/infrastructure/terraform
terraform apply -var-file="environments/dev.tfvars"
```
